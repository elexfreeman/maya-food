# Кулинарный сайт: Рецепты без лишних слов

## Project Overview
Это веб-сайт, посвященный кулинарии, с разделами: супы, салаты, гарниры, десерты и "мои рецепты". Сайт реализован с использованием HTML, CSS и простой структуры без JavaScript. Основная цель — предоставить пользователям рецепты, информацию о кулинарии и инструкции по здоровому питанию.

## Getting Started
### Prerequisites
- Браузер (Chrome, Firefox, Safari, Edge)
- Нет специальных требований — сайт работает в любом современном браузере.

### Installation
Сайт не требует установки — он готов к использованию. Все файлы находятся в корне проекта.

### Basic Usage
1. Откройте `index.html` в браузере.
2. Используйте меню в шапке для навигации по разделам.
3. Прокрутите страницу, чтобы увидеть рецепты и статьи.

### Running Tests
Нет тестов — сайт статичен и не требует запуска.

## Project Structure
- `index.html` — главный файл сайта с меню и контентом.
- `style.css` — стили для оформления.
- `logo.png` — логотип сайта.
- Подкаталоги (`supy`, `salaty`, `garniry`, `deserty`, `moi-recept`) — содержат рецепты, но не реализованы как отдельные страницы — они используются как ссылки в меню.

## Development Workflow
- **Coding Standards**: Используется простой HTML и CSS без сложных фреймворков.
- **Testing**: Нет тестов — сайт статичен.
- **Build and Deployment**: Нет сборки — сайт готов к размещению в любом хостинге.
- **Contribution Guidelines**: Для внесения изменений — редактируйте `index.html` и `style.css`, добавляйте новые разделы через меню.

## Key Concepts
- **Структура сайта**: Три колонки — левая (меню), центральная (содержание), правая (навигация).
- **Навигация**: Используется `id` для якорей и `scrollTo` для перехода.
- **Дизайн**: Минималистичный, с использованием цвета `#b56536` для ссылок и границ.

## Common Tasks
1. **Добавить новый рецепт**:
   - Откройте `index.html`.
   - Добавьте новый раздел в меню (в `header`).
   - Добавьте контент в центральную колонку.
   - Обновите `style.css`, если нужно изменить стили.

2. **Изменить дизайн**:
   - Редактируйте `style.css`.
   - Измените цвета, шрифты, размеры.

3. **Добавить новую категорию**:
   - Создайте новый каталог (например, `new-category/`).
   - Добавьте ссылку в меню (`header`).
   - Добавьте контент в `index.html`.

## Troubleshooting
- **Сайт не отображается**: Проверьте наличие файлов `index.html`, `style.css`, `logo.png`.
- **Ссылки не работают**: Убедитесь, что все `id` и `href` корректны.
- **Проблемы с дизайном**: Проверьте `style.css` на наличие ошибок.
